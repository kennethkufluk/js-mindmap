js-mindmap

This is a demo of js-mindmap. Sitemap items taken from www.digitas.com as an example.

  • Using an interactive mind map for blog post comments
    • Linear blog post comments don't represent input from people well
    • Interactive means your map changes as people make changes to the map
    • Needs to be fun and attractive, i.e., animated if possible
    • Permit representation of complex issues
      • Document all different angles to an argument
      • Allow people to follow different threads along different lines
      • Allow the editing and subsuming collections of arguments into new nodes
    • Implementation
      • Can't be a resource hog
